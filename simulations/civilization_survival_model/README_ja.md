# 文明生存モデル

このディレクトリには、異なる文明思想・哲学的前提のもとで、長期的な文明生存がどのように変化し得るかを比較するための簡易的な概念シミュレーションを収録しています。

本モデルは **予測用の科学モデルではありません**。自然補完科学の枠組みの中で議論を補助するための、透明性を重視したトイモデルです。数式とパラメータは説明用の仮定であり、検証済みの予測ではありません。

## シナリオ

- **Dualism / Human Supremacy（二元論・人間至上主義）**: 自然を外部・下位の対象として扱うため、技術能力が抽出圧を強めやすい。
- **Conventional Environmentalism（従来型環境管理）**: 抽出は抑制され、回復努力も存在するが、依然として人間中心の管理・資源利用の発想が残る。
- **Natural Complementary Science（自然補完科学）**: 技術能力が自然法則、調和、循環、構造、秩序、和に導かれ、回復能力とシステム持続性を高める方向に使われる。
- **Accelerated Extraction Collapse（崩壊加速型抽出）**: 短期的な抽出が強化され、生態・社会の基盤が急速に低下する。

## 追跡する変数

- `ecosystem_integrity`
- `circulation_health`
- `resource_base`
- `social_order`
- `technological_capacity`
- `civilizational_survival_index`

すべての値は 0 から 1 の範囲に正規化されています。

## 中核概念

このモデルは、技術がどの思想によって方向づけられるかの違いを表現します。

- 二元論・人間至上主義に導かれた技術は、抽出圧を高めやすい。
- 自然補完性に導かれた技術は、回復能力を高めやすい。
- 生態系完全性、循環の健全性、資源基盤、社会秩序が臨界水準を下回ると、崩壊リスクが高まる。
- 自然法則、調和、循環、構造、秩序、和を文明設計の原理として扱うほど、文明は長く存続しやすい。

## 結果の解釈

本モデルでは、自然補完科学モデルのみが長期的な文明存続度を維持する結果となった。

一方で、二元論・人間至上主義モデル、従来型環境管理モデル、加速的抽出崩壊モデルでは、時間の経過とともに生態系健全度、循環健全度、資源基盤、社会秩序が低下し、文明存続度も大きく低下した。

特に、自然循環の回復を文明設計に組み込まないモデルでは、約50年前後で文明存続度が急速に低下する傾向が示された。

これは未来予測ではない。
しかし、現代社会が気候変動、エルニーニョ、海洋循環の変動、森林火災、干ばつ、水不足、食料不安、生態系劣化といった複合的リスクに直面していることを考えると、文明の存続条件を経済成長や技術発展だけで考えることの危うさを示している。

本モデルにおいて唯一、長期的な存続傾向を示したのは、自然法則・調和・循環・構造・秩序・和を文明設計の中核に置く自然補完科学モデルであった。

この結果は、自然補完科学の主張を「理念」ではなく、文明存続条件を再設計するための概念的フレームワークとして示すものである。

## 実行方法

最小限の依存関係をインストールします。

```bash
pip install -r requirements.txt
```

シミュレーションを実行します。

```bash
python civilization_survival_sim.py
```

スクリプトは以下を保存します。

- `outputs/civilization_survival_results.csv`
- `outputs/civilization_survival_index.png`

## 範囲と制限

本モジュールは概念的・非予測的なものです。政策予測、生態予測、リスク定量化、科学的検証済みの主張として使用してはいけません。目的は、仮定を見える形にし、文明の価値体系が生態回復、資源安定性、社会秩序、長期的持続性にどのような影響を与え得るかを議論することです。
