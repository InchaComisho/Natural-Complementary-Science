# Repository Web Ring

## Overview

This document maps the network of related repositories in the InchaComisho framework and provides a suggested reading order.

Each repository addresses a different layer or dimension of the integrated framework.  
Together they form a coherent body of conceptual work that moves from:

```text
Philosophical foundation
    → AI evaluation and value alignment
    → Climate and ecological intervention
    → Causal analysis of the warming problem
    → Civilization design and implementation
```

All repositories present **conceptual, philosophical, or hypothetical frameworks** unless otherwise noted.  
Individual claims require independent scientific and engineering validation.

### Causal Analysis Layer

#### The-Real-Cause-of-Global-Warming-Not-Only-CO-Emissions-but-the-Collapse-of-Carbon-Fixation-Systems
**Role:** Structural analysis of why global warming is driven not only by emissions, but by the collapse of carbon fixation systems (terrestrial, oceanic, soil-based).  
**Contains:** Carbon fixation collapse hypothesis, quantitative scenario models, thermal inertia analysis, terraforming scale requirements.  
**Key argument:** The carbon equation has two sides; decarbonization addresses only one.  
**URL:** https://github.com/InchaComisho/The-Real-Cause-of-Global-Warming-Not-Only-CO-Emissions-but-the-Collapse-of-Carbon-Fixation-Systems

#### Why-Decarbonization-Alone-Cannot-Stop-Global-Warming
**Role:** Analysis of why decarbonization is necessary but structurally insufficient.  
**Contains:** Decarbonization limitations, thermal inertia dynamics, Announced vs. Realistic decarbonization scenarios, model limitations.  
**Key argument:** Emission reduction assumes healthy carbon sinks; without restoring those sinks, true carbon neutrality is structurally unachievable.  
**URL:** https://github.com/InchaComisho/Why-Decarbonization-Alone-Cannot-Stop-Global-Warming

## Note on Validation Status

All repositories in this web ring present **conceptual, philosophical, or hypothetical frameworks**.

- No physical intervention system described has been deployed at planetary scale
- No Artificial Wisdom system or Wa-Node network currently exists
- The causal claims about carbon fixation collapse require ongoing scientific investigation
- The New Civilizational Genesis Plan represents a long-term vision, not a validated implementation plan

These repositories are records of a structural framework and a civilizational hypothesis.  
Their value is in the coherence of the structural argument, not in claimed empirical certainty.
